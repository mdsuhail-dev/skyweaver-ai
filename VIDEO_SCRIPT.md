- Soft "ding" when result appears
- Subtle star twinkle sounds

**Visuals to Include:**
1. NASA night sky photos (royalty-free)

**Video Specs:**
- Duration: 2 min 20 sec
- Resolution: 1080p or 4K
- Format: MP4
- Aspect: 16:9
- Add captions for accessibility

**Free Tools to Create This:**
- OBS Studio (screen recording)
- CapCut (video editing - free, fast)
- DaVinci Resolve (advanced editing)
- Descript (AI-powered voiceover)

**Simple Script for Creating Video with CapCut:**
1. Record screen of working app (30 sec)
2. Add background music
3. Add captions/text overlays
4. Speed up some sections for pacing
5. Add transitions (keep it smooth/minimal)
6. Export as MP4
